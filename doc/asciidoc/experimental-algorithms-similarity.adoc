[[experimental-algorithms-similarity]]
= Similarity algorithms

[abstract]
--
This chapter provides explanations and examples for each of the similarity algorithms in the Neo4j Graph Algorithms library.
--

These algorithms help calculate the similarity of nodes:

* <<experimental-algorithms-similarity-jaccard, Jaccard Similarity>> (`algo.similarity.jaccard`)
* <<experimental-algorithms-similarity-cosine, Cosine Similarity>> (`algo.similarity.cosine`)
* <<experimental-algorithms-similarity-pearson, Pearson Similarity>> (`algo.similarity.pearson`)
* <<experimental-algorithms-similarity-euclidean, Euclidean Distance>> (`algo.similarity.euclidean`)
* <<experimental-algorithms-similarity-overlap, Overlap Similarity>> (`algo.similarity.overlap`)
